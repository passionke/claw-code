# Manager App

- วิธีดูภาพรวมร้านค้า Manager App

- วิธีดูรายงานผ่าน Manager App

- จัดการแก้ไขข้อมูลรายการสินค้า

- วิธีดูสต็อกสินค้าผ่าน Manager App

- เพิ่มสินค้าสินค้า (Manager)

- การเพิ่มรายการคอมโบ (Manager)

- เพิ่มหมวดหมู่สินค้าในเมนูลัดสินค้า

- การจัดการ ระบบPOS (Manager App)

- การจัดการ ระบบ Scan (Manager App)

- วิธีสร้างตัวเลือก/ตัวเลือกเพิ่มเติม (Manager App)

- เปิด-ปิด ร้านค้า GRABFOOD (Manager App)

- การจัดการรายการ GrabFood (Manager App)

- วิธีล็อกอินเข้าสู่ระบบ Manager App
