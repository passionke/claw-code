# การจัดการรายการ GrabFood (Manager App)

**Official docs:** https://gpos.co.th/th/user-manual/manager-app/managing-grabfood-items-via-manager-app

## Steps / Content

การจัดการ GrabFood ผ่านระบบนี้ช่วยให้ร้านค้าสามารถ ควบคุมสินค้าและราคาได้ตามช่องทาง จัดเรียงหมวดหมู่และเทมเพลตเมนูได้อย่างอิสระ ซิงค์ข้อมูลกับ POS ได้รวดเร็วและแม่นยำ

1. กดเลือก GrabFood

2. กด + นำเข้าสินค้า POS

3. กดเลือกหมวดหมู่ ก่อน จากนั้น กดเลือกรายการสินค้า ที่ต้องการโชว์บน Grab และกดตกลง

4.จากนั้นกดรูปปากกา เพื่อแก้ไข ข้อมูลรายการสินค้า จากนั้นกดบันทึก 

|สามารถแก้ไข ราคา , ตั้งเวลาการขาย และข่องทางการขายอื่นๆ

5.ระบบจะเด้งกลับมาหน้าเทมเพลตสินค้า และกดแก้ไข

5.กด เชื่อม เพื่อเลือกหมวดหมู่ จากสินค้าที่เพิ่มเข้ามา ใส่ระบบ Grab ก่อนทำการซิงค์ ข้อมูลไปยังหน้าการขาย GrabFood 

เมือเลือกหมวดหมู่ครบแล้ว กด ตกลง

6.จากนั้นเรียบร้อยแล้ว กดปุ่ม ซิงค์ เพื่อซิงค์ข้อมูลที่แก้ไข หรือเพิ่มใหม่ ไปยัง GrabFood

7. กด ... เพื่อเช็คตรวจสอบ ประวัติการซิงค์ หรือลบแทมเพลตได้เช่นกัน
