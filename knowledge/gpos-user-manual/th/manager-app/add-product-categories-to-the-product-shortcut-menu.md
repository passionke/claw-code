# เพิ่มหมวดหมู่สินค้าในเมนูลัดสินค้า

**Official docs:** https://gpos.co.th/th/user-manual/manager-app/add-product-categories-to-the-product-shortcut-menu

## Steps / Content

1. คลิกเลือก สินค้า

2. คลิกเลือก หมวดหมู่

3. กด เพิ่ม เพื่อกรอกชื่อหมวดหมู่ ส่วนชื่อสำรองหมวดหมู่ คือ ชื่อหมวดหมู่ ที่แสดงบน Grab

4. สามารถกดปุ่มสัญลักษณ์ ไอคอน เรียงลำดับ เพื่อกรอกตัวเลข จัดเรียงได้
