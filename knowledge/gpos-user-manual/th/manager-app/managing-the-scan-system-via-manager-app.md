# การจัดการ ระบบ Scan (Manager App)

**Official docs:** https://gpos.co.th/th/user-manual/manager-app/managing-the-scan-system-via-manager-app

## Steps / Content

1. คลิก สินค้า

2.กดเลือก ระบบScan

3. คลิก แก้ไข เทมเพลต

4. กด ... สามารถเลือก แสดงเป็นชุด เพื่อกดเปิดแสดงรายการบนระบบ Scan ที่ละหลายรายการ แต่หาก กดเลือก ซ่อนเป็นชุด จะเป็นการปิดไม่ให้รายการแสดงบนระบบ

5. หลังจากกดเลือก ใช้งานอย่างใดอย่างหนึ่งแล้วให้กดเลือกรายการสินค้าที่ต้องการ และกด บันทึก

6. จากนั้น รายการ จะเปิดแสดง หรือ ซ่อนไว้ ขึ้นอยู่กับเราเลือกใช้งาน เรียบร้อยแล้วกดบันทึกและออก

#### 6.1 แสดงเป็นชุด (สำหรับเปิดแสดงรายการ)

ระบบจะแสดงรายการสินค้าที่ทำการ ซ่อน ไว้ เพื่อทำการเลือกให้ แสดง บนระบบ Scan

- ให้ผู้ใช้งาน ติ๊กเลือกรายการสินค้า ที่ต้องการให้ แสดง บนระบบ Scan

- เสร็จแล้วกด "ตกลง" เพื่อยืนยันการแสดงสินค้า
