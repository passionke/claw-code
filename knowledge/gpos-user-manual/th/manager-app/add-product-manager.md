# เพิ่มสินค้าสินค้า (Manager)

**Official docs:** https://gpos.co.th/th/user-manual/manager-app/add-product-manager

## Steps / Content

1. คลิก สินค้า

2. คลิก เพิ่มสินค้า

3.กรอกชื่อสินค้า
 ด้านบน ชื่อแสดงบนระบบ POS 
 ด้านล่าง ชื่อแสดงบนระบบ Scan

4. Click the V button to select the category that has been created.

5. For the category, you can click Add to create a new category.

6. สินค้าไม่มีตัวแปร คือ สินค้าเดียว สามารถกรอกรายละเอียดรายการ เช่น ราคาขาย,ราคาต้นทุน และ หน่วยของสินค้า ฯลฯ

7. สินค้ามีตัวแปร คือ รายการสินค้าที่มีหลายราคา เช่น ชื่อรายการเสื้อ ตัวแปรเป็น S,M,L,Xl แยกราคา แยกบาร์โค้ดแต่ละตัวแปร

7. ตัวอย่างการสร้างรายการตัวแปร จากนั้นกด บันทึก

8. สินค้าชิ้น คือ หน่วยสินค้า สามารถกรอกชื่อลงได้ตามที่ร้านค้าเรียก เช่น ขวด แก้ว อัน ชิ้น ฯลฯ

9. ชั้งน้ำหนักสินค้า คือ หน่วยน้ำหนัก สามารถเลือกใช้งานได้ 2 รูปแบบ G,Kg ฯลฯ

10. ตัวเลือก/ตัวเลือกเพิ่มเติมสามารถสร้างก่อน ผ่าน คู่มือสร้างตัวเลือก 
กดเลือกตัวเลือกที่สร้างไว้ จากนั้น กด ตกลง

11.สามารถกำหนดราคาด้วยตนเองได้ที่หน้าเครื่อง ระบบ

11. สามารถเพิ่มรูปภาพอัพโหลดจากมือถือ

12. สามารถกดเลือก รูปร่างแทนการเพิ่มรูปภาพได้เช่นกัน
