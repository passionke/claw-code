# เปิด-ปิด ร้านค้า GRABFOOD (Manager App)

**Official docs:** https://gpos.co.th/th/user-manual/manager-app/open-close-grabfood-stores-via-manager-app

## Steps / Content

1. กดเลือก เดลิเวอรี่

2. กดเลือก สถานะ เพื่อกด เปิด & ปิดร้านค้า 

|ถ้าร้านค้าปิดอยู่ ระบบจะให้เลือก เปิดร้าน , หากระบบเปิดอยู่ ระบบจะให้เลือก ปิดร้าน

3. กดเลือก สถานะ ร้านค้า เปิดร้าน และกด บันทึก

4. กดเลือก เวลาทำการ ระบบจะสามารถตั้งค่า วันที่-เวลา กำหนด การเปิด-ปิดร้านค้า ตามช่วงวันและเวลาตามที่ต้องการได้

5. เมือตั้งค่าวัน-เวลา เปิด-ปิดร้านค้าเสร็จสิ้น ให้กดบันทึก จากข้อที่ 4 และกด ตกลง เพื่อยืนยันอีกครั้ง
