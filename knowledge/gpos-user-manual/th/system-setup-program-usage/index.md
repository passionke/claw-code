# เริ่มต้นระบบ & การใช้งานโปรแกรม

**Official docs:** https://gpos.co.th/th/user-manual/system-setup-program-usage

## Steps / Content

- การเชื่อมต่ออุปกรณ์และเปิดเครื่องใช้งานครั้งแรก
- สำหรับรุ่น D3 80mm

- สำหรับรุ่น D3 Pro

- สำหรับรุ่น D3 mini

- สำหรับรุ่น V3

- สำหรับรุ่น D3 Standard Single Screen

- การเริ่มต้นใช้งานซอฟต์แวร์ GPOS
- สำหรับเครื่องตั้งโต๊ะ

- สำหรับเครื่องพกพา

- การแก้ไขระบบ
- วิธีการแก้ไข IP Printer ด้วยโทรศัพท์มือถือ

- การใช้งานหน้าเครื่อง POS
- ตั้งสถาะการขายสินค้า

- แยกบิลรายการโต๊ะ
