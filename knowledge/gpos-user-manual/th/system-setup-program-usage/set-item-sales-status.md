# ตั้งสถาะการขายสินค้า

**Official docs:** https://gpos.co.th/th/user-manual/system-setup-program-usage/set-item-sales-status

## Steps / Content

ช่วยให้การทำงาน ตั้งค่าการเปิด ปิด เลิกขาย ได้เร็วและง่ายขึ้น ทำให้ครั้งละหลายรายการ สะดวกต่อการใช้งานสำหรับลูกค้าที่มีหลายรายการ

1.คลิกสีเหลี่ยมมุมซ้ายมือ

2. คลิกเลือก สถานะการขาย

3.คลิกเลือก รายการสินค้า และกดเลือกสถานะ เปิดขาย , สินค้าหมด หรือ ตั้งค่าจำนวนขายได้ ตามต้องการ เรียบร้อย กดตกลง
|ตั้งค่าจำนวนขายได้ คือ สตีอกของสินค้าที่ขายได้ หากหมด จะไม่สามารถขายสินค้าได้
