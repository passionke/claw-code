# แยกบิลรายการโต๊ะ

**Official docs:** https://gpos.co.th/th/user-manual/system-setup-program-usage/split-table-bill-items

## Steps / Content

เหมาะสำหรับกรณีนั่งโต๊ะรวมกัน แต่แยกจ่ายชำระ ผ่านรายการสินค้าบางรายการ ทำให้สะดวกต่อการแยกจ่ายตามความต้องการลูกค้า

1.คลิก สีเหลี่ยมด้านซ้ายมือบน > เลือกโต๊ะ

2.คลิก แยกบิล ด้านล่าง

3.คลิกเลือกโต๊ะประเภทสีแดง จากนั้น เลือกรายการที่จะชำระแยกออกมา เสร็จแล้ว คลิก
แยกการชำระเงิน

4.คลิก ยอดทั้งหมด เพื่อไปยังหน้าชำระเงิน

5.กด เก็บเงิน จากนั้น กดกลับสู่หน้าทำรายการ 
|ระบบจะเด้งกลับไปยังหน้าที่แยกรายการ

6.จากนั้น กดเครื่องหมาย X เพื่อลบออกและ กดยืนยันอีกครั้ง

7.คลิก ปิด เพื่อทำรายการชำระเงิน ยอดที่เหลือ

8.จากนั้น กดที่โต๊ะ ซ้ำอีกครั้งเพื่อไปหน้าแรกพร้อมชำระ

9.กด เก็บเงิน จากนั้น กดกลับสู่หน้าทำรายการ เพื่อชำระเงินสำเร็จ

## Keywords

แยกบิลรายการโต๊ะ, เหมาะสำหรับกรณีนั่งโต๊ะรวมกัน, แต่แยกจ่ายชำระ, ผ่านรายการสินค้าบางรายการ, ทำให้สะดวกต่อการแยกจ่ายตามความต้องการลูกค้า, คลิก, สีเหลี่ยมด้านซ้ายมือบน, เลือกโต๊ะ, แยกบิล, ด้านล่าง, คลิกเลือกโต๊ะประเภทสีแดง, จากนั้น
