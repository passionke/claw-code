# วิธีการใช้งานระบบทิป ช่องทางชำระเงินสด (หน้าเครื่อง)

**Official docs:** https://gpos.co.th/th/user-manual/device-settings/how-to-use-the-tipping-system-for-cash-payments-pos

## Steps / Content

วิธีการใช้งานระบบทิป เฉพาะช่องทางชํา ระ Cash (หน้าเครื่อง)1. กดที่ สี่เหลี่ยม ด้านซ้ายบน

2. เลือก “ทํา รายการ”

3. เลือก เมนู ที่จะทําการขาย

4. กด “ยอดทั้งหมด” ที่มุมขวาล่าง

5. เลือก ช่องทางชํา ระเงิน “Cash” เพื่อระบุจํา นวนเงินที่ได้รับ

6. ระบุ จํานวนเงิน ที่ได้รับ เพื่อให้ระบบคํานวนเงินทอน

7. กด “เก็บเงิน”

8. กด "เปลี่ยนมาเป็นทิป" ระบบจะนํา เงินทอนที่ต้องทอนทั้งหมดเปลี่ยนเป็นทิป

9. ยอดชําระจะเสร็จสิ้นโดยไม่ต้องทอนเงิน กด "กลับสู่หน้าทํารายการ" เป็นอันเสร็จสิ้น
