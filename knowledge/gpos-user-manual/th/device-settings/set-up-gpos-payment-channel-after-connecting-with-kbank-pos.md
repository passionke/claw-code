# ตั้งค่าช่องทางชำระเงิน Gpos หลังเชื่อมต่อ Kbank หน้าเครื่อง

**Official docs:** https://gpos.co.th/th/user-manual/device-settings/set-up-gpos-payment-channel-after-connecting-with-kbank-pos

## Steps / Content

1. กดที่ สี่เหลี่ยม ด้านซ้ายมือบน

2. ไปที่ “การตั้งค่า”

3. เลือก "ตั้งค่าประเภทการชํา ระเงิน"

4. หลังลิ้งค์ KBANK จะมีช่องทางชํา ระเงินเพิ่มขึ้นชื่อ “Prompt pay"

5. เลือก "ลําดับ"

6. กด ลูกศร ขึ้น/ลง เพื่อเลื่อนจัดลํา ดับที่ต้องการ

7. จัดลํา ดับเรียบร้อย ทําการกด “บันทึก”
