# วิธีการคืนเงินผ่านช่องทางชำระเงินอื่น (หน้าเครื่อง)

**Official docs:** https://gpos.co.th/th/user-manual/device-settings/how-to-refund-via-other-payment-channels

## Steps / Content

วิธีการคืนเงินผ่านช่องทางชํา ระเงินอื่น (หน้าเครื่อง)1. กดที่ สี่เหลี่ยม ด้านซ้ายบน

2. เลือก “จัดการออเดอร์และใบเสร็จ”

3. เลือก “คืนทั้งบิล”

4. เลือก “คืนเงินตามช่องทางเดิม”

5. ติ๊กที่ การคืนเงินตามช่องทางเดิม (ใช้วิธีการชําระเงินเดิมในการคืนเงิน)

6. ระบุ “เหตุผลในการคืนเงิน”

7. สามารถติ๊กที่

- พิมพ์ หากต้องการปริ้น

- สินค้าทั้งหมดถูกบันทึกเป็นขาดทุน สินค้าจะไม่ถูกบันทึกกลับเข้าสต๊อก
