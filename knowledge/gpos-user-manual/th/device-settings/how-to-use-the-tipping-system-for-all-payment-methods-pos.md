# วิธีการใช้งานระบบทิป ทุกช่องทางชำระเงิน (หน้าเครื่อง)

**Official docs:** https://gpos.co.th/th/user-manual/device-settings/how-to-use-the-tipping-system-for-all-payment-methods-pos

## Steps / Content

วิธีการใช้งานระบบทิป ทุกช่องทางชํา ระเงิน (หน้าเครื่อง)1. กดที่ สี่เหลี่ยม ด้านซ้ายบน

2. เลือก “ทํารายการ”

3. เลือก เมนู ที่จะทําการขาย

4. กด “ยอดทั้งหมด” ที่มุมขวาล่าง

5. กด “เพิ่มทิป”

6. เลือก ทิป ที่ทําการสร้างไว้ในระบบหลังบ้าน

7.1 เลือก “กําหนดทิปด้วยตนเอง”

7.2 ระบุ ยอดเงิน ที่ต้องการ

7.3 กด “เพิ่มทิป”

8. เลือก ช่องทางชําระเงิน แบบใดก็ได้ เนื่องจากมีการคํานวนทิปรวมในยอดชําระเรียบร้อยแล้ว

9. ระบุ จํานวนเงินที่ได้รับ กรณีเลือกช่องทางชําระ “Cash”

10. กด “เก็บเงิน”

11. กด "กลับสู่หน้าทํา รายการ" เป็นอันเสร็จสิ้น
