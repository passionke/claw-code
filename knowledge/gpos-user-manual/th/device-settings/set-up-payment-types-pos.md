# ตั้งค่าประเภทการชำระเงิน (หน้าเครื่อง)

**Official docs:** https://gpos.co.th/th/user-manual/device-settings/set-up-payment-types-pos

## Steps / Content

1. กดที่ สี่เหลี่ยม ด้านซ้ายบน

2. เลื่อนลงด้านล่าง เลือก "การจัดการตั้งค่าทั้งหมด"

3. เลือก "ตั้งค่าประเภทการชำระเงิน"

4. สามารถกดรูป "ปากกา" เพื่อทำการแก้ไขช่องทางชำระเงินที่สร้างไว้ หรือ กดรูป "ถังขยะ" เพื่อลบออกจากระบบ

5. กด "+ เพิ่ม" เพื่อสร้างช่องทางชำระเงินเพิ่มเติม

6. กรอก "รายชื่อชำระเงิน" ที่ต้องกาสร้าง

7. ติ๊ก เพื่อ เปิด/ปิด สถานะการใช้งานของช่องทางชำระเงินที่กำลังสร้าง

8. ติ๊ก เพื่อ เปิด/ปิด การแสดงรูปภาพบนหน้าจอฝั่งลูกค้าเมื่อชำระเงิน (รองรับตัวเครื่องแบบ 2 หน้าจอ)

9. หากเปิดการใช้งาน ระบบจะให้เพิ่มรูปภาพที่ต้องการแสดง สามารถกด "+ นำเข้า" เพื่อเพิ่มรูปภาพที่ต้องการ

- รองรับ jpg, png, jpeg, ขนาดภาพ: 6MB

10. ระบบจะขึ้นคิวอาร์โค้ด สามารถใช้มือถือสแกนเพื่อทำการอัพโหลดรูปภาพที่ต้องการ (ตามตัวอย่าง)

11. หลังจากเลือกรูปภาพที่ต้องการเรียบร้อยแล้ว (ตามตัวอย่าง) กด "ยืนยัน" เพื่อบันทึก

12. สามารถกด "+ อัปโหลดรูปภาพ" เพื่อเพิ่มไอคอนของช่องทางชำระเงิน

13. ระบบจะขึ้นคิวอาร์โค้ด สามารถใช้มือถือสแกนเพื่อทำการอัพโหลดรูปภาพที่ต้องการ (ตามตัวอย่าง)

14. หลังจากเลือกรูปภาพที่ต้องการเรียบร้อยแล้ว (ตามตัวอย่าง) กด "ยืนยัน" เพื่อบันทึก

15.“เลือกวิธีการคืนเงินที่รองรับ”ตามต้องการ

- หากเลือกการชําระเงินตามช่องทางที่สร้าง สามารถเลือกคืนเงินผ่านช่องทางชําระเดิมหรือ cash ได้ โดยระบบจะบันทึกเป็นข้อมูลการคืนเงินตามช่องทางนั้นๆ เท่านั้น

16. เสร็จแล้วกด "บันทึก" เพื่อยืนยัน

17. กด "ลำดับ" เพื่อทำการเรียงลำดับช่องทางชำระเงินที่ต้องการ

18. กด "ลูกศร ขึ้น-ลง" ที่ด้านหน้าช่องทางชำระเงินเพื่อเลื่อนขึ้นหรือลงตามต้องการ
