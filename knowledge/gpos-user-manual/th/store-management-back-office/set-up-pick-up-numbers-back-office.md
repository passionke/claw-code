# การตั้งค่าหมายเลข Pick up

**Official docs:** https://gpos.co.th/th/user-manual/store-management-back-office/set-up-pick-up-numbers-back-office

## Steps / Content

การตั้งค่าหมายเลข Pick Up นี้จะช่วยให้ผู้ใช้สามารถตั้งค่าและจัดการหมายเลขในระบบได้ง่ายและมีประสิทธิภาพ ลดข้อผิดพลาดและทำให้ระบบทำงานได้อย่างราบรื่นและยืดหยุ่นตามความต้องการ

### 1. เข้าสู่ระบบหลังบ้าน (Back Office)

ไปที่เว็บไซต์ login.gpos.co.th กรอก อีเมล และ รหัสผ่าน แล้วคลิกเข้าสู่ระบบ เมื่อเข้าสู่ระบบสำเร็จ เลือกเมนู "การจัดการตั้งค่าทั้งหมด"

### 2. เข้าสู่หน้าการตั้งค่าหมายเลข Pick Up

ภายในเมนู “การจัดการตั้งค่าทั้งหมด” ให้เลือกหัวข้อ “การตั้งค่า” จากนั้นเลือกเมนูย่อย “การตั้งค่าหมายเลข Pick Up”

## 3. กำหนดหมายเลขเริ่มต้นและสิ้นสุด

บนหน้าการตั้งค่าหมายเลข Pick Up คุณจะพบช่องสำหรับกรอกข้อมูลดังต่อไปนี้ :

- หมายเลขเริ่มต้น : กรอกหมายเลขที่คุณต้องการให้ระบบเริ่มนับ เช่น 1001

- หมายเลขสิ้นสุด : กรอกหมายเลขที่ต้องการให้เป็นจุดสิ้นสุด เช่น 1999

ตรวจสอบความถูกต้องของหมายเลขที่กรอก คลิกปุ่ม “บันทึก” เพื่อยืนยันการตั้งค่า

หมายเหตุ: เมื่อหมายเลข Pick Up ถึง “หมายเลขสิ้นสุด” ระบบจะวนกลับไปเริ่มที่ “หมายเลขเริ่มต้น” ใหม่โดยอัตโนมัติ หมายเลขรับสูงสุดไม่ควรน้อยกว่า 99

## Keywords

การตั้งค่าหมายเลข, Pick, นี้จะช่วยให้ผู้ใช้สามารถตั้งค่าและจัดการหมายเลขในระบบได้ง่ายและมีประสิทธิภาพ, ลดข้อผิดพลาดและทำให้ระบบทำงานได้อย่างราบรื่นและยืดหยุ่นตามความต้องการ, เข้าสู่ระบบหลังบ้าน, Back, Office, ไปที่เว็บไซต์, login, กรอก, อีเมล, และ
