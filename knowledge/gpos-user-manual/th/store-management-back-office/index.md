# การจัดการการตั้งค่าทั้งหมด (ระบบหลังบ้าน)

**Official docs:** https://gpos.co.th/th/user-manual/store-management-back-office

## Steps / Content

จัดการตั้งค่าทั้งหมดของระบบ ได้อย่างสะดวกผ่านระบบหลังบ้าน เพื่อให้เจ้าของร้านหรือผู้ดูแลสามารถปรับแต่งและควบคุมการทำงานของร้านค้าได้ง่าย และรวดเร็ว

- การจัดการภาษี/ประเภทชำระเงิน
- การจัดการค่าบริการเพิ่มเติม

- วิธีสร้างช่องทางชำระเงิน

- วิธีเพิ่มทิป

- การจัดการภาษี

- การเรียงลำดับช่องทางชำระเงิน

- การจัดการระบบโต๊ะ
- การเพิ่มโซน เพิ่มโต๊ะ

- การจัดการรูปแบบใบเสร็จ
- วิธีเพิ่มเครื่องพิมพ์ในห้องครัว

- ตั้งค่ารูปแบบใบเสร็จ

- การตั้งค่าเครื่องพิมพ์แคชเชียร์

- ตั้งค่ารูปแบบใบออเดอร์

- การตั้งค่ารูปแบบใบสั่งผลิตฉลาก

- ตั้งค่าแสดง QR Code ในใบเรียกเก็บเงิน หลังจากเชื่อมต่อ Kbank

- การจัดการร้านค้า
- การจัดการร้านค้า

- การตั้งค่า
- การตั้งค่ารุปแบบสต๊อก

- การตั้งค่ารูปภาพโฆษณาและหน้าจอหลัง

- การตั้งค่าช่องทางการขายในร้าน

- การตั้งค่าสแกนเพื่อสั่ง

- เลย์เอาต์การสแกนเพื่อสั่ง

- การตั้งค่าแจ้งเตือนสต๊อกสินค้า

- การตั้งค่าหมายเลข Pick up

- การตั้งค่าระบบคิว (ระบบหลังบ้าน)

- การตั้งค่าพื้นฐาน

## Keywords

การจัดการการตั้งค่าทั้งหมด, ระบบหลังบ้าน, จัดการตั้งค่าทั้งหมดของระบบ, ได้อย่างสะดวกผ่านระบบหลังบ้าน, เพื่อให้เจ้าของร้านหรือผู้ดูแลสามารถปรับแต่งและควบคุมการทำงานของร้านค้าได้ง่าย, และรวดเร็ว, การจัดการภาษี, ประเภทชำระเงิน, การจัดการค่าบริการเพิ่มเติม, วิธีสร้างช่องทางชำระเงิน, วิธีเพิ่มทิป, การเรียงลำดับช่องทางชำระเงิน
