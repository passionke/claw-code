# ระบบสำนักงานใหญ่

**Official docs:** https://gpos.co.th/th/user-manual/head-office

## Steps / Content

- การจัดการภาษี/ประเภทชำระเงิน สำนักงานใหญ่
- การสร้างเครื่องพิมพ์แคชเชียร์และส่งออกไปยังร้านค้า (สาขาย่อย)

- การสร้างเครื่องพิมพ์ในห้องครัวและส่งออกไปยังร้านค้า (สาขาย่อย)

- การสร้างทิปและส่งออกไปยังร้านค้า (สาขาย่อย)

- การสร้างค่าบริการเพิ่มเติมและส่งออกไปยังร้านค้า (สาขาย่อย)

- การสร้างประเภทการชําระเงินและส่งออกไปยังร้านค้า (สาขาย่อย)

- การจัดการรายการ สำนักงานใหญ่
- การลบรายการสินค้าจากสำนักงานใหญ่ไปสาขาย่อย

- การเปลี่ยนเมนูจากสำนักงานใหญ่ไปสาขาย่อย

- การอัพเดตรายการเดิมจากสำนักงานใหญ่ไปสาขาย่อย

- การเพิ่มรายการจากสำนักงานใหญ่ไปสาขาย่อย

- การตั้งค่าทั้งหมดในสำนักงานใหญ่
- การตั้งค่าและส่งออกไปยังร้านค้า (สาขาย่อย)

- การสร้างช่องทางการขายผ่านสำนักงานใหญ่

- สร้างเทมเพลตการตั้งค่าทั้งหมด (ระบบหลังบ้าน)

- เพิ่มร้านค้าสาขาย่อย

- สร้างพนักงานและผูกบทบาทจากสำนักงานใหญ่เพื่อดูแลร้านค้า (สาขา)

- การคัดลอกร้านค้า (ระบบหลังบ้าน)

- ตั้งค่าเครื่องพิมพ์ในห้องครัว และส่งออกไปยังร้านค้า (ระบบหลังบ้าน)

- ตั้งค่าทิป และส่งออกไปยังร้านค้า (ระบบหลังบ้าน)

- สร้างเทมเพลตตั้งค่าบิลใบเสร็จ บิลออเดอร์ และบิลเรียกเก็บเงิน (ระบบหลังบ้าน)

## Keywords

ระบบสำนักงานใหญ่, การจัดการภาษี, ประเภทชำระเงิน, สำนักงานใหญ่, การสร้างเครื่องพิมพ์แคชเชียร์และส่งออกไปยังร้านค้า, สาขาย่อย, การสร้างเครื่องพิมพ์ในห้องครัวและส่งออกไปยังร้านค้า, การสร้างทิปและส่งออกไปยังร้านค้า, การสร้างค่าบริการเพิ่มเติมและส่งออกไปยังร้านค้า, การสร้างประเภทการชําระเงินและส่งออกไปยังร้านค้า, การจัดการรายการ, การลบรายการสินค้าจากสำนักงานใหญ่ไปสาขาย่อย
