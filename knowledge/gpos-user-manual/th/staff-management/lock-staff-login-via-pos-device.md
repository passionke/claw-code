# วิธีการล็อกอินเข้าระบบของพนักงานผ่านหน้าเครื่อง pos

**Official docs:** https://gpos.co.th/th/user-manual/staff-management/lock-staff-login-via-pos-device

## Steps / Content

1. กรอก อีเมล และ รหัสผ่าน ของพนักงาน กดจํา รหัสผ่าน ก่อนเข้าระบบ

2. กดเลือก ร้านค้า

3. ป้อน รหัส PIN ที่ตั้งค่าไว้

4. เรียบร้อยแล้ว สามารถใช้งานขายได้เลย
