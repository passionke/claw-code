# วิธีเชื่อมต่อ Grab กับ GPOS (ดึงข้อมูลจาก Grab)

**Official docs:** https://gpos.co.th/th/user-manual/grab-x-gpos/connect-grab-with-gpos-sync-from-grab

## Steps / Content

คู่มือนี้ช่วยให้ร้านค้าสามารถเชื่อมต่อ GrabFood กับ GPOS ได้อย่างรวดเร็วและถูกต้อง ดึงข้อมูลสินค้าอัตโนมัติ ลดความผิดพลาด และเริ่มใช้งานระบบหลังบ้านได้ทันที

1. เข้าไปที่เว็บไซต์ login.gpos.co.th กรอก อีเมล และ รหัสผ่าน

การจัดการช่องทางGrabFood > GrabFood > ผูกบัญชี

หมายเหตุ : หากลูกค้ารายการสินค้า Grab และ Gpos เหมือนกัน แนะนำให้ลบรายการที่เหมือนกัน ออกจาก Gpos ก่อน ทำการซิงค์ ระบบ ป้องกันรายการสินค้านำเข้ามาซ้ำกัน

2. กรอก Gmail ของ Grab Merchant และคลิก ต่อไป

3. กรอก รหัสผ่าน Grab Merchant และคลิก ต่อไป

4. เลือกร้านค้าหรือสาขาที่ต้องการเชื่อมต่อกับ GPOS จากนั้นคลิก เชื่อมต่อ

5. ให้คลิก "ส่งออกเมนู" เพื่อดึงรายการสินค้าจากระบบ Grab เข้าสู่ Gpos 

|หากต้องการเปลี่ยนร้านค้า GrabFood สามารถคลิก เปลี่ยน ที่ด้านบนของหน้าจอ เพื่อเลือกร้านค้าใหม่ได้

6.คลิก เปิดการเชื่อมต่อ เพื่อทำการซิงค์ระบบ หน้านี้ใช้ระยะเวลาประมาณ 5 นาที
หมายเหตุ : ห้ามปิดหน้าจอหรือเปลี่ยนหน้าในระหว่างการซิงค์รายการ เพื่อป้องกันข้อมูลสูญหาย

7. เมื่อเชื่อมต่อเรียบร้อยแล้ว ให้ปิดหน้าต่าง Grab Merchant และเริ่มใช้งานผ่านระบบหลังบ้าน GPOS ได้ทันที
