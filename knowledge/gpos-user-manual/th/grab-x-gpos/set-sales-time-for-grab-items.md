# กำหนดเวลาการขายรายการสินค้า Grab

**Official docs:** https://gpos.co.th/th/user-manual/grab-x-gpos/set-sales-time-for-grab-items

## Steps / Content

สอดคล้องกับระบบ Grab ใช้งานร่วมกัน เมือมีการใช้งานร่วมกับ Gpos ช่วยให้ลูกค้าสามารถแก้ไข ดำเนินการ รายละเอียดเหมือนกับระบบ Grab

1.คลิกเลือก รายการ > รายการบน GrabFood > กดแก้ไขที่เทมเพลตเมนู

2.กดถัดไป เพื่อไปยังหน้ารายการสินค้า

3.กดเลือกปุ่ม V เพื่อเลือกตารางเวลาที่สร้างไว้อยู่แล้ว ถ้ากรณี ร้านเคยใช้งาน Grab อยู่ ระบบจะอ้างอิงข้อมูลจาก Grab ทั้งหมด

4.ถ้าต้องการสร้างใหม่ สามารถกด +เพิ่มเวลาที่จำหน่าย จากนั้นใส่ข้อมูลชื่อ วันที่และเวลา ที่ไม่ขาย หรือ หากไม่ต้องการกำหนดเวลา แนะนำให้กดเลือก ขาย 24 ชั่วโมง
|ช่องสุดท้าย จะกำหนดระยะเวลาหมดอายุการขายเปิดขาย ตามเงื่อนไขของระบบ Grab
