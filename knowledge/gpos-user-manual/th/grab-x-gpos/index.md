# Grab X Gpos

**Official docs:** https://gpos.co.th/th/user-manual/grab-x-gpos

## Steps / Content

- การเชื่อมต่อและการเพิ่มรายการ GrabFood
- วิธีเชื่อมต่อ Grab กับ GPOS (ดึงข้อมูลจาก Grab)

- เพิ่มรายการสินค้า GrabFood ผ่าน GPOS (ระบบหลังบ้าน)

- กำหนดเวลาการขายรายการสินค้า Grab

- วิธีเชื่อมต่อ GPOS กับ Grab (ดึงข้อมูลจาก GPOS)

- การใช้งานระบบ Grab บน Gpos
- วิธีแยกเมนู Grab และ POS ในร้านค้า (หลังบ้าน)

- วิธีตั้งราคาตัวเลือกที่ต่างกันจาก gpos และ grab จากหลังบ้าน
