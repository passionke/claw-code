# ระบบสมาชิก

- การจัดการสมาชิก
- ตั้งค่าพอยท์สมาชิก (ระบบหลังบ้าน)

- การจัดการระดับสมาชิก (ระบบหลังบ้าน)

- การเพิ่มสมาชิก (ระบบหลังบ้าน)

- การจัดการส่วนลดของสมาชิก
- แพ็กเกจเติมเงิน สำหรับสมาชิก

- ส่วนลดรายการแบบจำนวน (บาท) สำหรับสมาชิก

- ส่วนลดรายการแบบเปอร์เซ็นต์ สำหรับสมาชิก

- ตั้งค่าพอยท์สมาชิก ( หน้าเครื่อง)

- ระบบสมาชิก วิธีการเพิ่มสมาชิก (หน้าเครื่อง)
