# Add tables, add zones (POS)

**Official docs:** https://gpos.co.th/en/user-manual/device-settings/add-tables-add-zones-pos

## Steps / Content

1. กด สี่เหลี่ยมด้านซ้ายมือบน

2. เลือก จัดการโต๊ะ

3. เลือกโซนกดเพิ่ม

4. กรอก ชื่อโซน เช่น โซน A

5. เมื่อเพิ่มโซนครบแล้วเลือกโต๊ะกดเพิ่ม

6. กรอกชื่อโต๊ะจํานวนคนนั่งสูงสุดและลําดับโต๊ะ
