# Pickup settings (POS)

**Official docs:** https://gpos.co.th/en/user-manual/device-settings/pickup-settings-pos

## Steps / Content

### 1. เข้าสู่โปรแกรมขายหน้าร้าน

ไปที่โปรแกรมGPOS คลิกที่ไอคอนสี่เหลี่ยมมุมบนซ้ายเพื่อเปิดเมนูหลัก

### 2. ไปที่การตั้งค่าทั้งหมด

เลื่อนเมนูลงมาด้านล่างสุด เลือกเมนู: “การจัดการตั้งค่าทั้งหมด”

### 3. เข้าสู่หน้าการตั้งค่า Pickup

ไปที่เมนู “การตั้งค่า” เลือกหัวข้อ “การตั้งค่า Pickup”

### 4. กำหนดช่วงหมายเลข Pickup

ระบบจะขึ้นให้กรอก 2 ช่อง คือ :

- หมายเลขเริ่มต้น หมายเลขที่ระบบจะเริ่มใช้งาน

- หมายเลขสิ้นสุด หมายเลขสูงสุดที่ระบบจะใช้ ก่อนวนกลับไปเริ่มต้นอีกครั้ง

หลังกรอกหมายเลขครบทั้ง 2 ช่องแล้ว ให้กด “บันทึก” ระบบจะเริ่มต้นออกหมายเลข Pickup อัตโนมัติตามลำดับทุกครั้งที่มีออเดอร์ใหม่

หมายเหตุ: เมื่อหมายเลขถึงค่าที่กำหนดไว้ใน “หมายเลขสิ้นสุด”, ระบบจะ วนกลับไปที่หมายเลขเริ่มต้นอัตโนมัติ
