# Print barcodes via POS machine

**Official docs:** https://gpos.co.th/en/user-manual/device-settings/print-barcodes-via-pos-machine

## Steps / Content

1. กดที่ สี่เหลี่ยม ด้านซ้ายมือบน

2. เลือก "รายการ"

3. เลือก "พิมพ์บาร์โค้ด" ระบบจะขึ้น 2 ทางเลือก

4.1 เลือก "พิมพ์ทั้งหมด"

- ระบบจะปริ้นบาร์โค้ดของรายการที่ทํา การลงตัวเลขบาร์โค้ดไว้ทั้งหมดโดยจะไม่สามารถกดยกเลิกปริ้นได้

4.2 เลือก "พิมพ์บางรายการ"

5. เลือก หมวดหมู่ ที่ต้องการ

6. เลือก รายการสินค้า ที่ต้องการปริ้นบาร์โค้ด

7. กด "ปริ้น" ระบบจะทํา การปริ้นบาร์โค้ดตามรายการที่เลือก

8. หลังจากทํา การปริ้นเรียบร้อยแล้วกด "ปิด"
