# Set up QR Code display in Preliminary bill after connecting to Kbank

This manual is designed for store owners or system administrators who want to customize their receipts and order slips to reflect their brand identity, enhance professionalism, and make payment processes more convenient, according to the specific needs of their business.

### 1. Log in to the Back Office

Go to login.gpos.co.th , enter your email and password, and click Log In.
Once logged in, select the menu “Store Management” .

### 2. Access the Settings Menu

Go to “Print Management” and select “Receipt Management”.

### 3. Select the Receipt Template to Edit

Under the Preliminary Bill section, you can either use the default system preliminary bill template or click “+ Add a new invoice style” to open the preliminary bill design editor.

#### 3.1 Select the Sections to Display on the Preliminary Bill

You can view all available sections from the right-hand panel. The default sections include key fields suitable for different types of businesses. These predefined fields cannot be edited, except for “Custom Text” sections, which can be modified freely.

Main Sections :

- Basic Information

- Dish Information

- Cost Information

- Payment Information

- Member

- Footer

If your system is connected to KBank, a field called “Receipt Payment Code” will appear under the Footer. Click this field to make it visible on your printed receipt.

#### 3.2 Customize Using the “Property panel”

In the receipt editing screen, the Property panel allows you to adjust and design elements of your preliminary bill layout.

Available customization options include:

- Font Size – Adjust text size

- Bold – Emphasize text with bold styling

- Alignment – Align text to the left, center, or right

- Add – Insert new text, images, or dividers

- Blank line – Adjust spacing between lines

- Action – Move elements up or down, delete, or duplicate content

- Configurable – Allow or restrict editing from the POS terminal

Note : When adding new text, an input box will appear at the top of the Properties Panel for you to enter custom content.

#### 3.3 Add Images or Preview the Layout

You can select the preliminary bill size that best fits your printer model and preview the layout after customization. To add a logo or image, click “+ Upload” and choose the image file you want to use. When finished, click “Save” to store your customized receipt template.

Note : For best print results, use a black-and-white logo. Light or dark-colored images may be converted into darker grayscale tones when printed.

### 4. Activate the Customized Preliminary Bill Template

After saving, go to the “Preliminary Bill Template” list and select the design you created. Enable it as the default template — the system will use this layout automatically for every printed preliminary bill.

Additional options :

- Eye icon – Preview the created preliminary bill template

- Pencil icon – Edit the selected preliminary bill

- Trash icon – Delete templates that are no longer needed
