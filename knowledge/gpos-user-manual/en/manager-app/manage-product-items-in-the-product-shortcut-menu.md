# Manage and Edit Product Information (Manager)

**Official docs:** https://gpos.co.th/en/user-manual/manager-app/manage-product-items-in-the-product-shortcut-menu

## Steps / Content

1. Click Products.

2. Select Product List.

3. Click ...

Edit: Edit the existing product information.
Set Stock: Edit the stock quantity of the product.

4. Bulk Actions refer to editing, updating, or deleting multiple items at the same time.

5. Click Price Adjustment, select the products you want, then click OK.

6. Enter the new price in the price field, then click Save.

7. Enable/Disable: Select the product(s) and click OK.
Then choose whether you want to set them as Unavailable for Sale or Available for Sale.

8. Upload Image: Click and press the + button in the image field, then upload from your mobile phone gallery.

9. Click the pencil icon to edit the item name so it matches the product name in the system for the item you want to add an image to.

10. Click Upload
