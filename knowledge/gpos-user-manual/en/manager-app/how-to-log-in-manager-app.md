# How to log in Manager App

**Official docs:** https://gpos.co.th/en/user-manual/manager-app/how-to-log-in-manager-app

## Steps / Content

### 1. Download the Manager App GPOS

The app is available for both iOS and Android. Go to the App Store or Play Store and search for “GPOS”. Then, install the application. Once the installation is complete, open the app to begin using it.

### 2. Enter User Account Information

On the login screen, enter the email and password you used to register with the GPOS system. Then, check the boxes for

- Remember Password

- Accept Terms and Conditions

### 3. Log In

After verifying all information is correct, tap the “Log In” button to access the main screen of the app.
