# Connect Grab with GPOS (Sync from Grab)

**Official docs:** https://gpos.co.th/en/user-manual/grab-x-gpos/connect-grab-with-gpos-sync-from-grab

## Steps / Content

This guide helps merchants quickly and accurately connect GrabFood with GPOS, automatically import product data, reduce errors, and start using the back-office system immediately.

1. Go to login.gpos.co.th and enter your email address and password.

Go to GrabFood Channel Management > GrabFood > Link Account.

Note: If the customer’s product list on Grab and Gpos is the same, it is recommended to delete the duplicate items from Gpos first before syncing the system to prevent duplicate products from being imported.

2. Enter the Grab Merchant Gmail address and click Next.

3. Enter the Grab Merchant password and click Next.

4. Select the store or branch that you want to connect to GPOS, then click Connect.

5. Click “Export Menu” to import the product list from the Grab system into GPOS.

|If you want to change the GrabFood store, you can click “Change” at the top of the screen to select a new store.

6. Click “Enable Connection” to start syncing the system. This process takes approximately 5 minutes.

|Note: Do not close the screen or switch to another page during the syncing process to prevent data loss.

7. Once the connection is complete, please close the Grab Merchant window and start using the system directly through the GPOS back-office system immediately.
