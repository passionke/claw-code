# Set Option Price for GPOS & Grab (Back Office)

**Official docs:** https://gpos.co.th/en/user-manual/grab-x-gpos/set-option-price-for-gpos-and-grab-back-office

## Steps / Content

### 1. เข้าสู่ระบบหลังบ้าน (Back Office)

ไปที่เว็บไซต์ login.gpos.co.th กรอก อีเมล และ รหัสผ่าน แล้วคลิกเข้าสู่ระบบ
2. หลังจากล็อกอินเรียบร้อยแล้ว เลือก “รายการ”

3. เลือก “รายการบน GrabFood”

4. กดรูป ปากกา เพื่อแก้ไข

5. เลือก “การจัดการเครื่องปรุง”

6. เลือกกลุ่มเครื่องปรุงที่ต้องการปรับเปลี่ยนราคา ทำการตั้งราคาที่ต้องการ เสร็จแล้วกด “บันทึกเท่านั้น”

7. เลือก “รายการ” เลือก “รายการซิงค์” เพื่อบันทึกการใช้งานไปยัง GRAB
