# Connect Bluetooth Printer via USB

**Official docs:** https://gpos.co.th/en/user-manual/printer-settings/connect-bluetooth-printer-via-usb

## Steps / Content

1. กดที่ สี่เหลี่ยม มุมซ้ายบน

2. เลือก ‘เชื่อมต่อ’

3. กด ‘เพิ่มอุปกรณ์’

4. เลือกรุ่นเครื่องปริ้น ‘XP-58IIA’

5. เลือกวิธีการ เชื่อมต่อ ‘USB’ โดยที่เครื่องปริ้นมีการเสียบสาย USB กับเครื่อง POS เรียบร้อยแล้ว

6. เลือก ประเภทการพิมพ์ ที่ต้องการ

7. ตั้งชื่อ เครื่องปริ้น

8. เลือกความกว้างกระดาษ ‘58mm

9. กด ‘บันทึก’ ที่มุมขวาบน
