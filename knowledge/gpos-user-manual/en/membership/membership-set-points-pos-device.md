# Membership – Set Points (POS Device)

**Official docs:** https://gpos.co.th/en/user-manual/membership/membership-set-points-pos-device

## Steps / Content

1. กดสี่เหลี่ยมด้านซ้ายมือบน

2. เลือก “การตั้งค่าระบบสมาชิก”

3. จัดการพอยท์ กด “พอยท์ที่ได้รับ”

4. กด “เปิดใช้งานพอยท์ที่ได้รับ” กรอกจํา นวนเงินขั้นต่ํา ในการได้รับพอยท์

5. กดเลือก “กํา หนดอายุของพอยท์”

6. สามารถเลือกกํา หนดช่วงเวลา หรือ วันหมดอายุ วันที่ เดือน ระบบจะตัดหมดทุกปี

7. กดเลือก “เงื่อนไขการใช้พอยท์”

8. กรอก จํา นวนเงินที่มีค่า 1 พอยท์ เมื่อมีการใช้งานพอยท์

ลูกค้ายังได้รับพอยท์จากพอยท์ที่ใช้ชํา ระเงิน คือ

- กดเปิด หากบิลได้มีการใช้งานพอยท์ บิลนั้นจะยังสามารถได้รับพอยท์ตามที่ตั้งค่าไว้

- กดปิด ลูกค้าจะไม่ได้รับพอยท์ ในบิลที่มีการใช้งานพอยท์ สามารถได้รับพอยท์ได้เฉพาะบิลที่มีการชํา ระเงินปกติ
