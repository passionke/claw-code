# Top-Up Package for Members

**Official docs:** https://gpos.co.th/en/user-manual/membership/create-top-up-package-membership

## Steps / Content

This guide explains how to create and configure a Top-up Package Promotion for Members via the Back Office system, covering the steps for setting the promotion name, duration, member level conditions, and top-up amounts with bonus credits to ensure the promotion is created correctly and completely.

#### 1. Log in to the Back Office

Go to login.gpos.co.th, enter your email and password, then click Log In. Once logged in, navigate to Membership Marketing and select Promotion.

#### 2. Create a Top-up Package Promotion

The system will display the Promotions page. Under Top-up Promotion, click the “+ Create” button to begin.

#### 3. Enter the Promotion Name

The system will display a form for entering details. In the Promotion Name field, enter the desired name, for

example : February Top-up Package

Note : The name should be clear and descriptive, indicating the bonus amount or promotion period for easier management later.

#### 4. Set the Promotion Period

Select the period during which the promotion will be active by specifying the Start Date – End Date.

Example : The top-up package promotion is valid from February 1, 2026 – February 31, 2026.

#### 5. Set Member Level Conditions

Choose which member levels can use this promotion by selecting the appropriate option :

- All Levels – Members of all levels can use this top-up promotion.

- Specific Levels – Select only the member levels that are eligible for this promotion.

#### 6. Set Top-up Amounts and Bonuses

Specify the top-up package details by entering the Top-up Amount and the Bonus Amount the member will receive. Click “+ Add” to add another top-up level. Click “Delete” to remove a top-up level. Once all information is completed, click “Save” to confirm the promotion creation.

Example : Top up 1,000 THB, receive a 100 THB bonus.

Note : Once a top-up package promotion is created, it cannot be deleted from the system. Please carefully review all information before saving.
