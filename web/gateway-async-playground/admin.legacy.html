项目管理 · ds_1

在顶部 ds_id 下拉框切换项目；本页每 15s 静默同步当前项目配置（数据未变不重绘）。

Git 单向同步（DB → home/ → 远程）

每项目独立仓库（GitHub / GitLab HTTPS + PAT）。POST …/git/push 只推 home/ 中非 project_config 物化的文件（CLAUDE / skills / rules 等仍以 DB 为准，不进 Git）。

配置版本

状态机：① 全局至多 1 个临时版（各 Tab 保存只改它）② 生效只能是下表某一正式版（solve 用）③「保存为正式版」把临时版入库，不改生效 ④「设为生效」在正式版间切换 ⑤「废弃」删非生效正式版。

Project MCP

        

          MCP 仅存 PostgreSQL project_config.mcp_servers_json（无行则 solve 无 MCP）。
          下方「追加 / 全量替换 / 清空」即写库；也可用 PUT /v1/project/config/{ds_id}。
          GET /v1/mcp/injected/{ds_id} 仅探针，不返回 Bearer。
