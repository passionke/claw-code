项目管理

项目以 dsId 为键，配置落在 work_root/ds_{id}/home（Git 同步）。 PostgreSQL 仅存 gateway_sessions / gateway_turns / gateway_feedback，不存项目正文。

dsId环境就绪SkillsCLAUDE.mdworkDir
 

      


      

        
Project Skills

        
路径：ds_{id}/home/skills/<name>/SKILL.md。列表 GET /v1/skills/{ds_id}，保存 POST /v1/project/skills/{ds_id}。

        

          

          

            
            
            
            

              
              
            

          

        

      


      

        
Project MCP

        

          注入存网关内存并写入 .claw/settings.json。
          GET /v1/mcp/injected/{ds_id} 仅返回名称与探针结果，不返回完整配置；全量编辑请粘贴 JSON 后「全量替换」。
        

        

        
        
        

          
          
          
          
        

      


      

        
Project Rules

        

          对应项目规则文档 home/CLAUDE.md（网关 GET/POST /v1/project/claude/{ds_id}），与下方「系统提示词」不同：后者为运行时拼装后的完整 system prompt。
        

        
        

          
          
        

      


      

        
系统提示词（Effective Prompt）

        
GET /v1/project/prompt/{ds_id}/effective — 只读预览；POST 同路径触发重新加载。

        

          
          
        

        

        
      


      

        
Tools 配置

        

          存 project_config.allowed_tools_json（PUT /v1/project/config/{ds_id}）。
          勾选为空表示不额外限制，沿用网关全局 CLAW_ALLOWED_TOOLS。
